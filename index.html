🔗 Hubungkan ke Google Sheets
Ikuti langkah-langkah berikut, lalu paste URL Web App Anda di bawah ini.
  1. Buka Google Sheets yang ingin digunakan → klik menu Extensions → Apps Script
  2. Paste isi file Code.gs (yang sudah diunduh) ke editor, lalu klik Save 💾
  3. Pilih fungsi setupSheets dari dropdown → klik ▶️ Run → izinkan akses Google
  4. Klik Deploy → New deployment → Web App
    Set: Execute as: Me | Who has access: Anyone → Deploy
  5. Salin URL yang muncul (format: https://script.google.com/macros/s/.../exec) → paste di bawah
Setelah paste URL → klik Test Koneksi untuk memastikan terhubung, lalu Simpan & Sambungkan.
Untuk reset koneksi:
⚙️ Mode Offline (Demo): Data disimpan di browser lokal saja. Klik tombol ⚙️ Koneksi di pojok kanan atas untuk menghubungkan ke Google Sheets.
